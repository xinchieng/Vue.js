What is Vue.js?
===============
Vue.js 是一套开发Web网页的JavaScript脚本框架。

## 官方网站
* http://cn.vuejs.org/

## 竞争对手
http://cn.vuejs.org/v2/guide/comparison.html

## 技术前提
* HTML/CSS
* JavaScript
* Node.js（推荐）

## 开发工具
* 记事本等文本编辑器
  - Visual Studio Code
  - Brackets
  - ATOM
  * 浏览器（IE9，Chrome，FireFox，Safari）
